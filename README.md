# Engels Leren  📚

Een leuke app om Engelse woordjes te oefenen voor Unit 2!

## Installatie

### Vereisten
- Node.js (versie 18 of hoger) - Download van https://nodejs.org

### Stappen

1. **Open een terminal/command prompt** in deze folder

2. **Installeer de packages:**
   ```bash
   npm install
   ```

3. **Start de app:**
   ```bash
   npm run dev
   ```

4. **Open de app in je browser:**
   - Ga naar: http://localhost:3000
   - Of scan de QR code die in de terminal verschijnt met je iPad!

## Op je iPad gebruiken

Als de app draait op je computer:
1. Zorg dat je iPad op hetzelfde WiFi-netwerk zit
2. Zoek het IP-adres van je computer (staat in de terminal na `npm run dev`)
3. Open Safari op je iPad en ga naar: `http://[IP-ADRES]:3000`
4. Tik op het deel-icoon en kies "Zet op beginscherm" voor een app-icoon!

## Functies

- 🎓 **Leren** - Flashcards bekijken (beide richtingen)
- 🇳🇱➡️🇬🇧 **NL → EN** - Nederlands naar Engels oefenen
- 🇬🇧➡️🇳🇱 **EN → NL** - Engels naar Nederlands oefenen  
- ✍️ **Zinnen Maken** - Zelf zinnen schrijven met de woordjes
- 🏆 **Overhoren** - Test jezelf met beide richtingen door elkaar
- ⚙️ **Woordjes Beheren** - Woordjes toevoegen, bewerken of verwijderen

Succes met leren! 🎉
